# 9-valent-HPV-vaccination-and-head-and-neck-cancers

In this github analysis, all methods were identical to the publihsed paper 
https://jamanetwork.com/journals/jamaoncology/article-abstract/2847524
except for the date of extation (June 5, 2026) and the cohort extraction code which was identical to what Rhodes et al indicated.
The figure showed that the coding difference proposed by Rhodes et al had no impact on the study findings.
As the TriNetX platform is a live database which may create a slightly different result based on the timing of data extraction.

Figure shows "Cumulative incidence of HPV-related cancer among males aged 9–26 years by the exact coding proposed by Rhodes"
Cumulative incidence is presented by 9-valent HPV vaccination status. HPV indicates human papillomavirus.
